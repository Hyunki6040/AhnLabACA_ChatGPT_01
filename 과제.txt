챗봇 제작에 있어서 다음과 같은 내용을 구현하라.

1. 프리랜서 가이드라인(출판본).PDF 데이터 기반 챗봇
- 전체 글에 대한 요약본을 볼 수 있도록 하라.
  예) 이 책을 전체적으로 요약해줘.
- 전체 책의 정보를 알 수 있도록 하라
  예) 이 책은 총 페이지지?
  이 책의 저자는 누구야?  
- 전체 글에 대한 예상 질문을 볼 수 있도록 하라.
- 전체 목차를 확인할 수 있도록 하라.
  예) 전체 목차를 출력해줘.
  프리랜서의 미래는 몇 페이지에 있지?
- 책 내용에 대한 질문에 대한 답변은 반드시 몇 페이지에 근거가 있는 지 밝혀라

2.OutdoorClothingCatalog_1000.csv 기반 챗봇
- 전체 상품 정보에 대한 개괄을 안내할 수 있도록 하라.
  예) 처음 접속시 "안녕하세요. 아웃도어 전문 매장입니다. 현재 저희는 xxx 제품을 다루고 있으며 필요한 제품을 자동으로 안내해드리고 있습니다 등"
- 상품에 대한 전반적인 통계 정보를 안내하라.
  예) 총 상품수, 성별 의류수, 계절 별 의류수, 의류종류별 의류수 등
- 제품 추천 시 제품 번호를 아울러 출력하라 

3. 공통 사안
- 사용자 별로 히스토리를 관리하라. 히스토리는 총 5개 질문까지만 저장해도 된다.
- 프론트엔드는 html 버전과 command line interface 중 하나를 선택하여 개발하라.
- 디자인은 자신의 포트폴리오를 위해서 가능하면 아름답고 편리하게 구현하라.
- 소스코드는 깃헙의 자신의 레포지토리에 업로드하고 공개하라.
  ( 단 API key 값 공개는 절대 금지 )
- 앞의 1,2번 데이터의 챗봇 구현은 별도로 하거나 같이 하거나 상관없음.
